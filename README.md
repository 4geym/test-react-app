# First big project on React for education

The project contains such things as:
-controlled input;
-sorting;
-Search;
-memoization;
-modal window;
-animations;
-decomposition;
-custom hooks;
-work with the Axios server;
-loading indication;
-pagination;
-dynamic navigation;
-authorization.
